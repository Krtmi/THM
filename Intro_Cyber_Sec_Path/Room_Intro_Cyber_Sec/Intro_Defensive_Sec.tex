\subsubsection*{Intro to Defensive Security}

\T{Introduction to Defensive Security}{
Defensive Security deals with the preventative part of the security architecture and the response measures after an instrusion, as well as with the awareness of the users.\\
In this room, we cover:\\
\begin{itemize}
\item Security Operations Center (SOC)
\item Threat Intelligence
\item Digital Forensics and Incident Response (DFIR)
\item Malware Analysis
\end{itemize}
}

\T{Areas of Defensive Security}{
Two main topics of defensive security are Threat Intelligence, organised as a Security Operations Centre (SOC) and Digital Forensics and Incident Response (DFIR), as well as Malware Analysis.\\
\textbf{Security Operations Center (SOC):}\\
Cybersecurity team that monitorsthe network to detect anomalous and potentially dangerous cyber security events.\\
They focus on:\\
\begin{itemize}
\item Vulnerabilities: fix them with the appropriate update or patch. If not possible, prevent damage coming from an exploit.
\item Policy violations: look after the abiding of all rules necessary or established.
\item Unauthorized activity: log and monitorize unexpected events to block any potential damage.
\item Network intrusions: detect and prevent damage coming from it.
\end{itemize}
}
